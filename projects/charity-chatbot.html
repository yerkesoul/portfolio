Despite being thousands of kilometers away, I had the incredible opportunity to contribute to a heartfelt charity initiative by developing and deploying a chatbot for a project in Astana. The initiative, named Arman, was organized by my cousin, Dana Yessimkhanova, and Akerke Korganbekova, as part of their annual tradition of finding "Secret Santa"‚Äîor "Deduska Moroz" (Grandfather Frost) in post-Soviet countries‚Äîfor children with special needs and those in orphanages.

Supporting the project remotely from Germany, I created a chatbot to automate the process of matching children‚Äôs wish-filled letters with volunteers eager to fulfill them. It was incredibly rewarding to contribute to this meaningful initiative and help bring joy to so many children.

This architecture illustrates a cloud-based system using AWS for event or task management, referred to as "Arman." It integrates AWS services and the Telegram API for communication. Here's an overview:

Components and Workflow:

  • Telegram API:

    Handles user interactions via messages or updates.

  • EC2 Instance:

    Hosts application logic, processes requests from Telegram, and interacts with the database and storage.

    Backed by EBS for persistent storage.

  • SQL Database:

    Stores structured data like user info and event details.

  • S3 Bucket:

    Stores unstructured data (e.g., logs, reports) and feeds the dashboard.

  • Dashboard for Organizers:

    A web app that visualizes data from the S3 bucket for monitoring and management.

  • VPC:

    Ensures secure communication between components.

Data Flow:

  • Users interact via Telegram, with requests processed by the EC2 instance.
  • The EC2 instance fetches or updates data in the SQL database.
  • Logs/reports are stored in the S3 bucket for dashboard visualization.
  • The VPC ensures security and efficient communication.

The Goal

The goal of Arman Chat Bot was to streamline the process of assigning children‚Äôs letters‚Äîfilled with their wishes‚Äîto dedicated volunteers.

How the Chatbot Worked:

  1. A user learns about the rules of Arman and then enters their name and phone number.
  2. They select a number between 1 and 275.
  3. The bot shares details about the chosen child, including their wish, a phone number, and a link to their letter in PDF format. For example, I have attached one letter which belongs to an 11-year-old boy named Sultan, whose wish was to receive LEGO from Deduska Moroz.
